Рөл туралы (About the Role)

Сіз біздің басты өніміміз — "Atameken OS" тактикалық басқару интерфейсін жасаумен айналысасыз. Бұл жай ғана веб-сайт емес, бұл операторларға нақты уақыт режимінде ондаған автономды активтерді (дрондар, роботтар) басқаруға мүмкіндік беретін күрделі инженерлік интерфейс.

Негізгі басымдық — деректерді визуализациялау. Сіз WebGL және 3D технологияларын қолданып, ұрыс алаңының цифрлық көшірмесін және тактикалық карталарды интерфейске интеграциялайсыз.

Сіздің міндеттеріңіз (Responsibilities)

  • React және TypeScript негізінде масштабталатын және өнімді интерфейстерді әзірлеу.
  • Mapbox GL немесе Three.js технологияларын қолданып, 3D карталарды және тактикалық нысандарды визуализациялау.
  • WebSocket арқылы келетін телеметриялық деректерді (кідіріссіз) өңдеу және көрсету.
  • Күрделі деректерді визуализациялау үшін оңтайландырылған Dashboard-тар мен графиктік компоненттерді құру.
  • UX дизайнерлермен бірлесіп, стресстік жағдайдағы операторларға арналған ыңғайлы интерфейс (UI) жасау.
  • Код сапасын жақсарту және тестілеу (Unit, Integration tests).

Талаптар (Requirements)

  • React.js және TypeScript бойынша 3+ жылдық тәжірибе.
  • WebGL, Three.js немесе ұқсас 3D графикалық фреймворктармен жұмыс тәжірибесі.
  • State management (Zustand, Redux Toolkit немесе MobX) құралдарын жетік білу.
  • Заманауи CSS (Tailwind CSS, Styled Components) және адаптивті дизайн принциптерін түсіну.
  • Нақты уақыттағы деректермен (WebSockets) жұмыс істей білу.
  • Интерфейс өнімділігін (Performance optimization) талдау және жақсарту дағдылары.

Артықшылық болады (Nice to have)

  • Географиялық ақпараттық жүйелермен (GIS) жұмыс тәжірибесі.
  • Canvas API немесе SVG анимацияларын терең білу.
  • Дизайн құралдарымен (Figma) жұмыс істей алу.

Командаға қосылуға дайынсыз ба?

Түйіндемеңізді (CV) және портфолиоңызды (немесе Github) жіберіңіз.
